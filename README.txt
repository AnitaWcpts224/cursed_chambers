CursedChambers.py


System/Run requirements:
-Any system should work but this game was developed/tested on Windows 10
-Requires Python 3, preferably more recent version
-This game uses the 'pygame' module version=2.0.0


Controls/Gameplay:
-Move your charachter with the arrow keys.
-Fire with the spacebar.
-Avoid being hit by enemy mobs/bullets, while destroying the enemy with your own bullets.
